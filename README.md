# Godart-corentin Devjournal

## How do I install these formulae?

`brew install godart-corentin/devjournal/<formula>`

Or `brew tap godart-corentin/devjournal` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "godart-corentin/devjournal"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
